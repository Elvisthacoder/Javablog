Hello World Example
===================

This example demonstrates how easy it is to create a Hello World example 
by using Scooter framework.

--------------------------
How to run the application
--------------------------

Run the app with default port 8080
scooter>java -jar tools/server.jar examples/greeting

Then browse http://localhost:8080/greeting

Please do browse http://localhost:8080/greeting/examples
