PetClinic Application
=====================

This example demonstrates how easy it is to create a petclinic application 
by using Scooter framework.

--------------------------
How to run the application
--------------------------

First make sure database is created. The database script for MySQL is in 
static/docs directory.

Then run the app with default port 8080
>java -jar tools/server.jar examples/petclinic

Now browse http://localhost:8080/petclinic
